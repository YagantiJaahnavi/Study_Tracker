 Project Overview

Study Tracker is a simple Java-based console application that helps users track their daily study time and compare it with a target study goal. It uses basic Java concepts like control flow and file handling to store and manage study data.

Features

Take daily study time input from user
Set and compare with target study hours
Show progress (below, equal, or above target)
Track study data day by day
Store data using a text file (day.txt)
